Item Check · now automatic

The verdict comes from the real pickit.

Hover an item in game, press Ctrl+C, then open Item Check. The item is normally already pasted and checked—no Ctrl+V and no extra button. If clipboard access is blocked, paste it and choose Check this item. You see the deciding rule, the generated .ipd line, and what to change when the answer is no.

Not a simulation. Item Check runs the exact same pickit-generation code that writes the file. For the same current settings, the tab and the file cannot disagree.

Fractured items stay practical: the result shows the actual target mods so you can compare them with the item in front of you.

Picked up

See whether the item cleared your price floor, appears on the always-take list, or belongs to a category that ignores the floor—plus the emitted rule.

Ignored

See exactly why it missed and the setting that changes the result.

Worth 6.20 ex, but your item floor is 8 ex—it misses by 1.80 ex. Fix: Lower the item floor to 6.2 ex or less and it gets taken.

Depends on the rolls

For covered rare gear, the bot does the maths in-game, so Item Check shows the scored stats and threshold instead of inventing a yes. If no recipe covers the base or the slot is off, the verdict is a definitive no.

Good to know

Straight answers before you install.

What does it actually create?

A complete Exiled Bot 2 pickit (.ipd) plus an optional matching in-game loot filter. Generate again whenever you want current market prices.

I am on v4.20.0 or v4.21.0. Why will the updater not work?

Those two releases have a broken update path. Close the app, download v4.29.2 manually once and open it. Your settings, profiles and bot folder remain in place; in-app updates work normally after that.

Why might Windows warn about the download?

It is a free unsigned community app, so SmartScreen may ask you to confirm. The source, build workflow and SHA-256 checksum are public. The app only reads public price/update data and never asks for your game account.

What are the value floors?

One floor covers currency and item categories; the other covers unique gear. Anything below its floor is skipped. Presets and Auto-floor set sensible values if you do not want to tune them manually.

How does rare gear scoring work?

Each of the 17 slots has a weighted recipe. A rare is kept only when its useful stats clear that slot's score, using verified stat IDs and real top rolls from game data.

What if a game patch breaks a rule?

The app refreshes its game data and runs a health check against live mod tables. Dead or renamed rules are named instead of silently failing.

Do I need to reload the optional in-game filter?

Yes. After every Generate, reselect it in Options → Game → Filters. This does not apply to Exiled Bot's .ipd pickit.

My bot walked past an item—what should I check?

Hover the item in game, press Ctrl+C, then open Item Check. It normally reads the copied item and checks it automatically; if clipboard access is blocked, paste it and choose Check this item. If the verdict is Picked up but the bot still ignores it, run the connection check to confirm pickit.ini's active_profile matches the generated filename.

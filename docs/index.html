See on the ground what the pickit selects.

Before you download

Straight answers, without the wall of text.

What does the app create?

It can generate a complete Exiled Bot 2 pickit (.ipd) from live prices and write a matching in-game loot filter. Create your filter can also make a .filter from any existing Exiled Bot pickit.

Can I safely convert someone else's pickit?

Yes. The source pickit is only read, never modified, and nothing is uploaded. The report lists rules read, converted, shown wider, and untranslatable, with a line and reason for anything that cannot convert.

Should I leave Hide everything else on?

It starts on for a clean filter preview, and gold is never hidden. Turn it off while botting because hidden ground labels can stall pickup. If any rule cannot translate, the app shows a loud warning before you rely on the filter.

Why might Windows warn about the download?

It is a free unsigned community app, so SmartScreen may ask you to confirm. The source code, build workflow, and SHA-256 checksum are public. The app never asks for your game account.

I use v4.20.0 or v4.21.0. How do I update?

Close that version, download v4.35.2 manually once, and open it. Your settings, profiles, and Exiled Bot folder remain untouched. Later in-app updates work normally.

Item Check says Picked up, but the item was missed. What should I check?

Run the connection check and confirm pickit.ini's active_profile matches the generated filename. Item Check shows what the generated pickit says; the connection check confirms Exiled Bot is reading that same file.

Do I need to reload the optional in-game filter?

Yes. After every Generate, reselect it in Options → Game → Filters. This does not apply to the generated Exiled Bot .ipd pickit.

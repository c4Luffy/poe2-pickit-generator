Know what you get

Two files. Two different jobs.

The pickit tells Exiled Bot 2 what the generated rules select. The optional loot filter controls what you see on the ground in Path of Exile 2.

.ipd

Bot pickit

Used by Exiled Bot 2. Your pickit.ini needs to point to this generated filename.

.filter

In-game loot filter

Used by Path of Exile 2. Select it again under Options → Game → Filters after every save or regeneration.

v4.41.21Current release · 23 July 2026

Prices are ready before you ask.

The price fetch now starts in the background at launch, so the Economy tab opens instantly instead of making you wait on it.

  • v4.41.21 · Prices load in the background — opening Economy used to fetch 24 separate poe.ninja price lists on the spot, five at a time, each with a back-off wait whenever poe.ninja rate-limited, while you watched “Loading prices…”. That same fetch now starts in the background shortly after the app opens, so the tab is normally populated the moment you click it. Everything that reads prices shares one 15-minute cache, so Generate, the Chance tab and Auto-floor get the same head start. It is fire-and-forget: offline or failed, the tab loads exactly as before, and Refresh prices still forces a live re-fetch. Read the v4.41.21 notes.
  • v4.41.20 · Economy tab overhaul — hover any item for a card showing its art, live price, 7-day trend, keep/skip status and the exact pickit rule that catches it; click a row to pin the card (a Copy button and its own keep/skip toggle built in), or right-click a row to copy its rule instantly. Value bars behind each price read as relative worth on a log scale, the sidebar groups (General, Equipment, Atlas, Always pick) collapse so the category list fits without scrolling, and clicking a row no longer toggles a rule off by accident. The Economy tab is faster too, and Generate now shows a start and finish toast. Read the v4.41.20 notes.
  • v4.41.19 · Tablets are priced live, not hardcoded — poe.ninja now prices Precursor Tablets (Overseer, Abyss, Breach, Ritual, Irradiated, Temple, Delirium, each per rarity) and Unique Tablets, so generated pickits respect the normal value floor for them like any other market item instead of force-picking every rarity. Some are genuinely valuable — a Normal Ritual Tablet has been worth close to a Divine. Both appear as their own Economy categories under Atlas, matching poe.ninja’s own layout, with Precursor Tablets grouped by type. Waystones moved to Always pick, since poe.ninja never prices them. Read the v4.41.19 notes.
  • v4.41.18 · Four audit findings — standard formatted rules and exchange pick-all rules now escape literal quote marks, but quote escaping is still incomplete elsewhere in the current source; no current cached item contains a quote. The app and saved-settings generation paths now keep always-kept names active even if poe.ninja ever prices one through a unique-shaped payload, while the legacy --cli path still lacks that protection. Static Economy views no longer paint a sort arrow that does nothing. Both real writers now include the existing Exiled Bot 2 syntax guide as comments in generated .ipd files. Read the v4.41.18 notes.
  • v4.41.17 · Rare-gear strictness you can tune — built across v4.41.14, v4.41.15 and v4.41.17, rare gear still uses the same per-slot recipes and stat weights, but its score bar is no longer fixed. Choose Looser, Balanced, Strict or Very strict for all 17 slots, then override any slot on its own; Balanced remains the tuned default. Each slot shows an example from its own stats, such as “a T1 Life + T1 Spirit + T1 Fire res rare scores about 300, clears 250, kept.” Read the v4.41.17 notes.
  • v4.41.16 · Regenerate from saved settings without opening the window — run python -m exilebot_pickit --regenerate to rebuild the pickit from your saved floors, switches, global and per-slot rare strictness, output folder and auto-copy setting. Add --league “Name” to override the saved league, or use the command with Task Scheduler to keep the generated pickit current on a timer. Read the v4.41.16 notes.

Straight answers

Before you download.

Is this the bot?

No. It is a generator for Exiled Bot 2 pickit files and optional Path of Exile 2 loot filters.

Can I convert a pickit made somewhere else?

Yes. Drop any Exiled Bot .ipd into the app. The source stays read-only, and the report explains anything the game filter cannot express exactly.

What happens when a rule cannot be translated exactly?

The report marks it as Shown wider or Untranslatable, then gives the source line and reason. Review every warning before relying on Hide everything else.

Why was an item missed when Item Check says Picked up?

Item Check reports what the generated pickit says. Run the connection check and confirm that pickit.ini points to that same generated file.

What does “Depends on the rolls” mean?

The generated pickit covers that rare base, but Exiled Bot scores the item’s actual modifiers in game. Item Check shows the scored stats and threshold instead of inventing a final answer.

Should I leave Hide everything else on?

It starts off and remembers your choice; gold is never hidden. Keep it off while botting because hidden labels can stall pickup.

Why didn’t my in-game filter update after regeneration?

Path of Exile 2 may still have the previous filter loaded. After saving or regenerating, select the .filter again under Options → Game → Filters.

Where are generated files saved, and are backups made?

The app writes the generated .ipd and optional .filter to your chosen output folder. Rotating backups protect generated output before replacement and can be restored from Settings.

Why does Windows show a warning?

The app is not code-signed. Its source, build workflow, and SHA-256 checksum are public, so the downloaded executable can be verified.

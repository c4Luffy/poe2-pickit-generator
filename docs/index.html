On first launch the app looks for your Configuration\default\Pickit folder and, if it finds one, turns on Auto-copy for you.

4

⚠️ Tell the bot which file

Open the bot's pickit.ini and set active_profile=poe2_pickit — it must match the app's output name.

Why step 4 matters more than the rest: Exiled Bot only ever loads the one .ipd named by active_profile. Get it wrong and auto-copy still reports success, the file really does land in the folder — and the bot carries on reading whatever old pickit it was already on. Nothing looks broken. Nothing you generate is ever used. The app's connection check catches this for you and offers a one-click fix.
FAQ

How it works & good to know.

How does it actually work?

Pick your league and press ⚡ Generate. The app pulls live poe.ninja prices, works out what's worth money right now, and writes a full Exiled Bot 2 pickit (.ipd) — 2,000+ rules — plus a matching in-game loot filter. Point the bot at the file once, and every future Generate refreshes it on its own.

What are the value floors?

Two numbers. The first covers everything that isn't unique gear — currency, runes, essences, fragments, tablets, chance and craft bases. The second covers unique gear. Anything below its floor is left on the ground.

Uniques sit higher because cheap ones drop constantly and just eat stash tabs. Not sure where to start? Pick a preset — or turn on Auto floor and it reads the current economy for you.

Do I have to find the bot folder myself?

No. On first launch the app looks for your Exiled Bot install and, if it finds one, sets the folder and turns on Auto-copy for you. There's an Auto-detect button in Settings to re-run it. It only accepts a folder that has the bot's own pickit.ini beside it, so it won't grab the wrong one.

How does it decide a rare is worth keeping?

Each of the 17 gear slots has a scoring recipe: a rare's stats are added up by weight, and it's kept only if the total clears a threshold. Every weight comes from that stat's real top roll in the game data, and every scored stat is checked against the bot's own mod list — no guesswork.

Do I need to reinstall when the game is patched?

No. Unique prices are pulled live on every Generate, and the bundled base and category lists update themselves from GitHub at launch. A game-data health check also flags when a patch has renamed or removed something your rules rely on.

My bot isn't picking things up

Nine times out of ten it's step 4: open the bot's pickit.ini and check active_profile matches the app's Output file name. The bot loads only the file named there.

Then, in order: did you press Generate after changing anything? Is Settings → Game data health clean — a patch can silently kill rules. And is your currency floor set so high it's binning runes and essences?

What's the .ipd vs the .filter?

The .ipd is the pickit the bot reads — that's the one that matters. The .filter is a matching in-game loot filter for manual play, so what the bot values and what you see on the ground line up.

Is it safe? Windows flagged the download

It's an unsigned free tool, so Windows shows a caution — click More info → Run anyway. The full source is public on GitHub, and it only ever talks to poe.ninja and GitHub.

Does it play the game or trade for me?

No. It only writes the pickit file (and an optional loot filter). It never touches your account — prices are read-only from poe.ninja.

Up and running in 4 steps

The last one is the step everybody misses — and it's the difference between the bot using your pickit and quietly ignoring it.

Download & open

Grab the .exe and double-click. Windows warning? → More info → Run anyway.

Pick league, Generate

Choose your current league and press ⚡ Generate. Live prices in, pickit out.

Point the app at the bot

On first launch the app finds the bot's Configuration\default\Pickit folder for you and turns Auto-copy on. Not found? Hit Auto-detect in Settings.

⚠️ Tell the bot which file

Open the bot's Configuration\default\pickit.ini and set active_profile=poe2_pickit — it must match the app's output name.

Why step 4 matters: Exiled Bot only ever loads the one .ipd named by active_profile. Get it wrong and auto-copy still says “success”, the file really does land in the folder — and the bot carries on reading whatever old pickit it was already on. Nothing looks broken, and nothing you generate is ever used. The in-app 📖 Setup guide walks you through all four steps.

How it works & good to know

How does it actually work?

Pick your league and press ⚡ Generate. The app pulls live poe.ninja prices, works out what's worth money right now, and writes a full Exiled Bot 2 pickit (.ipd) — 2,000+ rules — plus a matching in-game loot filter. Point the bot at the file once, and every future Generate refreshes it on its own.

Do I have to find the bot folder myself?

No — on first launch the app looks for your Exiled Bot install (Desktop, Downloads, Program Files, the drive roots…) and, if it finds one, sets the folder and turns on Auto-copy for you. There's an Auto-detect button in Settings to re-run it any time. It only accepts a folder that has the bot's own pickit.ini beside it, so it won't grab the wrong one.

What are the value floors?

Two numbers on the Generate tab — one for unique gear, one for everything else (currency, runes, bases…). Anything below the floor is still written into the pickit but commented out, so the bot skips it. Raise a floor to be pickier, lower it to grab more. Not sure where to start? Turn on Auto floor and it sets them from the current economy.

How does it decide a rare is worth keeping?

Each of the 17 gear slots has a scoring recipe: a rare's stats are added up by weight, and it's kept only if the total clears a threshold. Every weight comes from that stat's real top roll in the game data, and every scored stat is checked against the bot's own mod list — no guesswork. Toggle any slot on or off in the Magic & Rare tab.

Do I need to reinstall when the game is patched?

No. Unique prices are pulled live on every Generate, and the bundled base/category lists update themselves from GitHub at launch. A game-data health check in Settings also flags when a patch has renamed or removed something your rules rely on.

What's the .ipd vs the .filter?

The .ipd is the pickit the bot reads — that's the one that matters. The .filter is a matching in-game loot filter for manual play, so what the bot values and what you see on the ground line up.

Is it safe? Windows flagged the download

It's an unsigned free tool, so Windows shows a caution — click More info → Run anyway. The full source is public on GitHub, and it only talks to poe.ninja and GitHub.

No prices are loading

Open the Debug tab → Run API test. If poe.ninja is down, the app falls back to its saved price cache and keeps working.

My bot isn't picking things up

Nine times out of ten it's step 4: open the bot's Configuration\default\pickit.ini and check that active_profile matches the app's Output file name (poe2_pickit by default). The bot loads only the file named there — if it doesn't match, Auto-copy still says “success” and the bot keeps reading an old pickit.

Then, in order: did you press Generate after changing anything (Preview warns when it's showing an old run)? Is Settings → Game data health clean — a game patch can silently kill rules? And is your Currency & items floor set so high it's binning runes and essences?

Does it play the game or trade for me?

No. It only writes the pickit file (and an optional loot filter). It never touches your account — prices are read-only from poe.ninja.
